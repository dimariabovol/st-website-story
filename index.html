rem/em

rem – относительно корневого элемента <html> (16px)

em – относительно текущего/родительского элемента

                .btn {
                  font-size: 1rem;
                 
                  min-width: 6em;
                  padding: 2em;
                  margin: 1em;
                }
→ →

Что это даёт?

  • не теряется соотношение между отступами в блоке
  • не теряется соотношение расстояния между блоками
  • элемент масштабируются пропорционально

Вертикальный ритм
Vertical rhythm

Достигается с помощью общего знаменателя для создания отступов.

Как реализовать?

  1. Устанавливать свойства размера шрифта кратно 0.5rem (8px)
  2. Устанавливать отступы между элементами кратно 0.5rem (8px)

Ключевое слово "кратно".

            .text {
              font-size: 1rem;
              line-height: 1.5rem;
            }
        
            .box {
              margin-bottom: 0.5rem | 1rem | 1.5rem... ;
            }

UI Kit

UI Kit – набор элементов, на которых строится UI сайта.

Cодержит все элементы, которые пользователь видит на ресурсе.

UI KIT - Studytube website

Для чего это нужно?

  • упрощения и комплексного подхода к созданию проекта
  • позволяет быстро создавать каждый следующий модуль/страницу
  • является документом проекта, с которым сверяются на каждом этапе работы

Fallback

Когда пользователь это увидит?

  • пока фоновая картинка не загрузится полностью
  • если изображения не доступны (несовпадение регистра/адреса)
  • если показ картинок в браузере отключен
